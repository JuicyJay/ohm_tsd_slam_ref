Constructor & Destructor Documentation

ohm_tsd_slam_ref::SlamNode::SlamNode ( void  )

Constructor.

class can be called in 2 modes, runSlam or runLocalize (run method invoked by start method) default mode: SLAM mode, generates new empty obvious::TsdGrid, dimension and resolution can be set by defines in SlamNode.h LOCALIZE mode: loads content from file at a given path, operates on a fixed model; in the fixed model no push into the grid is needed, no thread mapping (?richtig?) object is instantiated
